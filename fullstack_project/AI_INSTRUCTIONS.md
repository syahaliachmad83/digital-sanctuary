Jika Anda menggunakan ekstensi AI seperti Continue di VS Code dengan model Gemini:

Buka panel chat AI di samping.

Ketik @gemini.md (AI akan membaca file tersebut sebagai konteks).

Lalu ketik instruksi Anda, misalnya: "@gemini.md Buatkan endpoint CRUD untuk data Produk, lalu buatkan halaman React-nya."

Hasilnya? Saya (atau model AI manapun yang Anda gunakan) akan langsung membuatkan file di folder app/api/v1/endpoints/products.py, menggunakan skema Pydantic, dan membuatkan antarmuka React dengan Tailwind CSS yang rapi, tanpa Anda harus mengulangi instruksi panjang lebar!
